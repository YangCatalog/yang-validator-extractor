About

This application is written using the python Django framework using a combination of xym to fetch and extract YANG modules from IETF specifications, pyang to validate the extracted modules, the confdc YANG validator and compiler from the ConfD management agent toolkit, the yanglint YANG data modeling language parser and toolkit from CESNET and the yumawork`s application yangdump-pro - YANG language parser. The web application code is available in github under a BSD 3-Clause License. If you have feedback, feature requests or a bug to report, please open an issue on github.
